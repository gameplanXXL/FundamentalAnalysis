# analyst-sidecar

This folder stores persistent memory for the **Viktor** (Fundamental-Analyst) agent.

## Purpose
Viktor merkt sich Watchlists, Analyse-Historie und User-Präferenzen über Sessions hinweg, um personalisierte und kontextbezogene Analysen zu liefern.

## Files
- **memories.md** — User-Profil, Watchlist, Analyse-Historie, erkannte Muster
- **instructions.md** — Protokolle, Verhaltensregeln, Analyse-Standards, Easter Eggs

## Runtime Access
After BMAD installation, this folder will be accessible at:
`{project-root}/_bmad/_memory/analyst-sidecar/`
